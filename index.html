Conhecimentos

HTML

A linguagem principal da web, ela permite que os desenvolvedores desenhem como serão exibidos pelo navegador os elementos da página, como texto, hiperlinks e arquivos multimídia.

CSS

O CSS define como serão exibidos os elementos contidos no código de uma página da internet e sua maior vantagem é efetuar a separação entre o formato e o conteúdo de um documento.

JavaScript

JavaScript é uma linguagem de programação de alto nível criada, a princípio, para ser executada em navegadores e manipular comportamentos de páginas web.

Git e GitHub

GIT é utilizado para a criação de um histórico de alterações em código-fonte de projetos de desenvolvimento de software. Github é um repositório remoto, criado como um serviço online de hospedagem de repositórios do Git.

React

O React é a biblioteca mais popular do JavaScript e é usada para construir uma interface de usuário (IU). Ela oferece uma resposta excelente para o usuário adicionar comandos usando um novo método de renderizar sites.

Material UI

É um conjunto de diretrizes, também conhecido como guidelines, que foi criado pela Google para padronizar todas as suas interfaces gráficas.

Figma

Figma é um editor online de design gráfico focado na criação de interfaces gráficas e experiência de usuário com ênfase colaborativa.

Formação em ADS

Análise e desenvolvimento de sistemas. Graduação de nível tecnológico que torna o estudante apto a projetar, testar, implantar e gerenciar sistemas e recursos que envolvem Hardware e Software.
